# Pokemon App

## Demo

https://pokemon-app-eight-cyan.vercel.app/

## Features

- Show list of pokemon
- Show detail pokemon
- Catch pokemon with chance rate 50%
- Give a name to pokemon you caught
- Show list of my pokemons that have been caught
- Delete pokemon
